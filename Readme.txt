Для создания структуры таблиц в MS SQL Server:

1. Откройте директорию проекта в комманднойй строке(cmd, far, Visual Studio...).
2. Запустите комманду - "dotnet ef migrations add Initial --context ApplicationDbContext"
3. Запустите проект на выполнение, база заполнится тестовыми данными, которые будут отображены на первой странице приложения.
